# Portfolio Website

This is a static website inspired by [portfoliohere.co.uk](https://www.portfoliohere.co.uk/).

## Project Structure

```
def-not-an-art/
├── css/
│   └── style.css
├── images/
│   └── (placeholder images)
├── js/
│   └── script.js
├── index.html
└── README.md
```

## Features

- Responsive design
- Portfolio grid layout
- Mobile-friendly navigation
- Smooth animations

## Usage

1. Clone this repository
2. Replace placeholder images with your own artwork
3. Customize text and content in `index.html`
4. Deploy to your hosting provider of choice

## Image Requirements

For the portfolio grid to look its best, use images with the following characteristics:
- Consistent aspect ratio (recommended: 4:3 or 3:2)
- High resolution (at least 800px wide)
- Optimized for web (JPG or WebP format)
